# Xamsa Game Terms

## LEVELING SYSTEM

### XP Points

XP are granted for content creation. So, when users create packs, topics, questions, etc. or host games, they will earn XP points. Each user start from 0 XP. XPs are always increasing, cannot be decreased.

#### XP Levels

XP levels are the following:

- 0 XP -> 99 XP [NEWBIE]
- 100 XP -> 249 XP [ADEPT]
- 250 XP -> 999 XP [VANGUARD]
- 1000 XP -> 4999 XP [ELITE]
- 5000 XP -> 9999 XP [ASCENDANT]
- 10_000 XP -> 19_999 XP [MYTHIC]
- 20_000 XP -> 49_999 XP [PARAGON]
- 50_000 XP -> 99_999 XP [LEGEND]
- 100_000 XP -> 999_999 XP [EXALTED]
- 100_000 XP -> 999_999 XP [IMMORTAL]

## Game Level Terms

- _MVP_: The winner of the game
- _Silver_: 2nd place of the game
- _Bronze_: 3rd place of the game
- _Wooden Spoon_: The last ranked player of the game
- _Annihilator_: Winner with at least 2x more points than the second place player
- _Survivor_: Winner with less than 500 points difference from the second place player
